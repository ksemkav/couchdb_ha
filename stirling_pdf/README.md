# Stirling PDF Add-on

## About

Stirling PDF is a powerful, locally hosted web-based PDF manipulation tool that allows you to perform various operations on PDF documents. This Home Assistant add-on brings comprehensive PDF functionality directly to your smart home environment, enabling you to manage, edit, and transform PDF files without relying on external services or compromising your privacy.

## Installation

1. Navigate to your Home Assistant interface
2. Go to **Settings** → **Add-ons** → **Add-on Store**
3. Click the **⋮** (three-dot menu) in the top-right corner
4. Select **Repositories**
5. Add this repository URL and click **Add**
6. Return to the Add-on Store and locate **Stirling PDF**
7. Click on the add-on and press **Install**
8. Wait for the installation to complete
9. Enable **Start on boot** and **Watchdog** (optional but recommended)
10. Click **Start** to launch the add-on

## Usage

Once the add-on is running:

1. Click **Open Web UI** in the add-on page, or navigate to the add-on's port in your browser
2. The Stirling PDF interface will load with all available tools organized by category
3. Select the desired PDF operation from the menu
4. Upload your PDF file(s) and configure the operation parameters
5. Process and download the result

The add-on runs entirely locally, ensuring your documents remain private and secure.

## Configuration

This add-on works out of the box with no required configuration. All settings can be managed through the Stirling PDF web interface.
